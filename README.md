# Multi-Cloud Cost Comparison and Optimization Analytics Dashboard

An interactive analytics solution designed to track, compare, and optimize infrastructure costs across multiple cloud services providers, including AWS, Azure, and Google Cloud Platform (GCP).

## 🚀 Key Features
* **Multi-Cloud Tracking:** Consolidated view of resource consumption and billing data from AWS, Azure, and GCP.
* **Cost Optimization Analytics:** Key insights to identify underutilized resources and suggest cost-saving opportunities.
* **Interactive Visualization:** Dynamic filtering by cloud provider, region, service type, and billing period.

## 🛠️ Built With
* **Microsoft Power BI Desktop:** For data modeling, relationship mapping, and dashboard creation.
* **Microsoft Excel:** As the core structured data repository for cloud billing metrics.
